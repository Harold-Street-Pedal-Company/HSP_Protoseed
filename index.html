HSP Protoseed

By Harold Street Pedal Company

The HSP Protoseed is a $50 CAD prototyping platform for Daisy Seed–based guitar pedals, designed and milled in Canada. Each unit is home-milled with care, not mass-produced. You’re getting a robust, ready-to-use PCB plus a growing open-source library that makes learning and building pedals fast and frustration-free. We’re also releasing our own DSP-based effects for free to the platform, so you’ll always have fresh projects to explore.

Why HSP Protoseed?

  • Made in Canada with home-milled quality and character.
  • Includes the HaroldPCB library – get up and running with minimal boilerplate.
  • Beginner-friendly examples for both mono and stereo audio processing.
  • Expandable for advanced DSP work – from breadboard testing to production builds.
  • Supports all core Harold Street pedal hardware: pots, toggles, footswitches, LEDs.

Quickstart

  1. Clone the repo: git clone https://github.com/Harold-Street-Pedal-Company/HSP_Protoseed.git
  2. Install the HaroldPCB library into your Arduino libraries folder.
  3. Open one of the example sketches in Arduino IDE.
  4. Flash to your Daisy Seed and start experimenting.

Library Features

  • Simple StartAudio() for stereo or mono callbacks.
  • Easy reading of pots, toggles, and footswitches.
  • Built-in DSP helpers: filters, limiters, and utility classes.
  • No unnecessary dependencies – optimized for DaisyDuino.
